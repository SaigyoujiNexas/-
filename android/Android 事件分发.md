# Android 事件分发

**事件只有一个**

|     事件      |             简介              |
| :-----------: | :---------------------------: |
|  ACTION_DOWN  |       初次碰到屏幕触发        |
|  ACTION_MOVE  | 屏幕上滑动时触发， 会多次触发 |
|   ACTION_UP   |        离开屏幕时触发         |
| ACTION_CANCEL |       事件被上层拦截时        |

Android提供了两套事件处理机制

* 基于监听的事件处理
* 基于回调的事件处理
