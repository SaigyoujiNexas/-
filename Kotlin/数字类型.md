## 数字类型

所有数字类型都有符号

Kotlin提供了很多安全转换函数

```kotlin
fun main() {
    val number1: Int = "8.98".toInt()	//numberFormatException
}
```

```kotlin
fun main() {
    val number1: Int? = "8.98".toIntOrNull()
    println(number1)	//print null
}
```

## Double 转Int

```kotlin
    println(8.956756.toInt())		//8
    println(8.956756.roundToInt())	//9
```

```kotlin'
    val format = "%.2f".format(8.956756)
    println(format)		//8.96
```
